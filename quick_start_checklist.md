# 🚀 Quick Start Checklist
## Get Your Portfolio Live in 30 Minutes

---

## ⏱️ 30-Minute Setup

### Step 1: GitHub (10 minutes)

**1.1 Create Account**
- [ ] Go to github.com
- [ ] Click "Sign Up"
- [ ] Choose username (e.g., `abongile-tyenjana`)
- [ ] Verify email

**1.2 Create Repository**
- [ ] Click "+" → "New repository"
- [ ] Name it: `portfolio`
- [ ] Description: "Business Intelligence Portfolio - Power BI"
- [ ] ✅ Public
- [ ] ✅ Add README
- [ ] Click "Create repository"

**1.3 Upload Files**
- [ ] Click "Add file" → "Upload files"
- [ ] Drag these files:
  - `portfolio.html` (rename to `index.html`)
  - `README.md`
  - `dashboard_template_guide.md`
  - `implementation_steps.md`
  - `quick_reference.md`
- [ ] Commit message: "Initial portfolio upload"
- [ ] Click "Commit changes"

**1.4 Enable GitHub Pages**
- [ ] Go to "Settings"
- [ ] Click "Pages" (left sidebar)
- [ ] Source: **main** branch, **/ (root)**
- [ ] Click "Save"
- [ ] Wait 2-3 minutes
- [ ] Your site is live at: `https://yourusername.github.io/portfolio/`

---

### Step 2: Customize Portfolio (10 minutes)

**2.1 Edit index.html** (in GitHub)
- [ ] Click on `index.html`
- [ ] Click pencil icon (Edit)
- [ ] Update Line 46-49: Your email, LinkedIn, GitHub URLs
- [ ] Update Line 98: Your "About Me" text
- [ ] Click "Commit changes"

**2.2 Edit README.md**
- [ ] Click on `README.md`
- [ ] Click pencil icon
- [ ] Replace `yourusername` with your actual username
- [ ] Update email and LinkedIn
- [ ] Click "Commit changes"

---

### Step 3: LinkedIn (10 minutes)

**3.1 Update Profile**
- [ ] Headline: "Business Intelligence Analyst | Power BI Specialist"
- [ ] About section: Add portfolio URL
- [ ] Featured section: Add link to portfolio
- [ ] Skills: Add Power BI, DAX, Data Analysis

**3.2 Create Announcement Post**
```
🚀 Excited to share my Business Intelligence Portfolio! 

I've built an interactive portfolio showcasing my Power BI work - 
featuring a comprehensive sales analytics dashboard with 26+ 
advanced DAX measures and time intelligence.

🔗 Check it out: https://yourusername.github.io/portfolio/

#PowerBI #DataAnalytics #BusinessIntelligence #PortfolioShowcase
```

- [ ] Copy template above
- [ ] Update your GitHub Pages URL
- [ ] Add a screenshot if possible
- [ ] Post!
